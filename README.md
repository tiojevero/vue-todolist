# Vue Todo List with Drag & Drop

### Stack
| JS Framework     | Vue 3      |
|------------------|------------|
| CSS Framework    | Tailwind   |
| State Management | Vuex       |
| Bundler          | Vite       |

### How to run
```bash
  pnpm install
  pnpm run dev
```
